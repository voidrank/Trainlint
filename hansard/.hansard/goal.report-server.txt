Host every trainlint plugin installer's interactive reports at https://secondfoundationlabs.com, each user behind their own login, with no always-on server or tunnel anywhere. Each installer's plugin uploads its rendered report to a Cloudflare Worker, which stores it in R2 namespaced by that installer's identity; Cloudflare Access gates the site and the Worker serves each user only their own reports. The three pillars: per-user-isolation (tenant-binding), serverless-store-and-serve (R2 + Worker), and plugin-upload. DONE = an installer installs the plugin, logs in once to get an upload token, and thereafter every report is viewable at secondfoundationlabs.com behind their own login, provably isolated from every other tenant, with zero always-on infra on their machine or mine.
